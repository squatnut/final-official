.
|-- final

### Instructions

- `$ git clone https://github.com/wsu-cs3620/final.git`
- `$ cd src/public/`
- `$ vim info.php` just look over the file
- `$ vim index.php` just look over the file
- `$ cd ../..`
- `$ ./start-webapp.sh`
- Go to http://localhost/hello/world
- `$ ./stop-webapp.sh` when complete
